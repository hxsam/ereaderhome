EReaderHome

This project aims to create an Android launcher that makes ebooks the primary focus of your android tablet (or phone)

Milestones for the project

  1. create initial UI
    • book list
    • book details
  2. read in ebooks from a root directory
  3. watch the ebook directory structure for changes
  4. import metadata from epub's
  5. import metadata from mobi's
  6. import metadata from pdf's
  7. implement a cover flow UI
